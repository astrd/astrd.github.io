Astrid Gamoneda

Hi there! I’m a software engineer interested in decentralization and allowing people to control their own privacy. I started studying computer science in 2016 because I enjoy solving challenges and fixing problems. When the Cambridge Analytica data scandal broke I was shocked and I started getting more and more interested in privacy and digital ownership so when I was presented with an oportunity to work on a project that aims to decentralize the internet I did not hesitate to join. This resulted in my thesis project OHMYPOD.

Relevant projects

  • OHMYPOD!

    The aim is to develop a proof-of-concept demonstrating that there exists a way to use modern web services, for instance, browsing in an e-commerce site, without giving out control of one’s own data using solid technology and decentralization. This project was developed by WESO (Web Sematic Research Group of the University of Oviedo) working with empathy.co, an e-commerce search company.
    Available here:

  • A decentralised chat: Sole Chat

    Team project building a decentralised messaging application using the solid framework. This results in improved privacy for users and true data ownership
    Available here:

  • Foon Robotics Project

    The aim of this project is to structure knowledge representation for robot learning and manipulation using directed graph with object and motions where the edges denote a sequencing of actions called foon (Functional Object-Oriented Network). Developed in Java.
    Available here:

  • K-nearest Neighbour

    An Implementation of the knn machine learning algorithm in python to be executed with pyspark. Working with two dataset (the NASDAQ stock and Breast Cancer Winconsin dataset). This was a group project. Developed in pyhton
    Available here:
